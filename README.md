# Chunibyo Dimension

A neon anime-inspired 3D website built with Three.js and GitHub Pages.

## Features

- 🌌 Animated 3D universe
- 🌀 Rotating TorusKnot model
- ✨ Neon Chunibyo aesthetic
- ⭐ Floating stars in 3D space
- 💎 Glassmorphism overlay UI
- 🎌 Anime-inspired design
- 📱 Responsive fullscreen layout
- ⚡ GitHub Pages compatible

## Technologies

```txt
HTML • CSS • JavaScript • Three.js • WebGL
```

## Theme

Inspired by anime Chunibyo aesthetics:

- glowing magic energy
- dimensional gateway vibes
- fantasy UI effects
- purple neon atmosphere
- dramatic anime-style presentation

## GitHub Pages

Enable GitHub Pages:

```txt
Settings → Pages → Deploy from branch → main → /root
```

After enabling Pages, the site will automatically go live.
