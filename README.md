# 3D Website

A futuristic 3D website built with Three.js and GitHub Pages.

## Features

- 🌌 Animated 3D background
- ⭐ Interactive space scene
- 🌀 Rotating 3D torus
- 💎 Modern glassmorphism UI
- 📱 Responsive design
- ⚡ Fast static hosting
- 🌐 GitHub Pages compatible

## Technologies

```txt
HTML • CSS • JavaScript • Three.js • WebGL
```

## GitHub Pages

Enable GitHub Pages:

```txt
Settings → Pages → Deploy from branch → main → /root
```

Then your website will be live.
